.AutoDash is a web application created with the Framework of Django as a back-end, a front-end using HTML, CSS, Bootstrap and a little bit of JavaScript. Additional libraries in Python helped to achieve this project such as Pandas, Dash and PyCaret.

With a user-friendly interface, .AutoDash makes it easy to understand and analyze your data. Additionally, the application also you to apply machine learning techniques to your data, providing even more insights and understanding. Whether you're a student, researcher, or data professional, .AutoDash is the perfect tool for exploring and understanding your data.
